infraudit

Linux server security auditing from the command line.

A single binary you drop on any Linux server to audit its security posture. Validates users, SSH, firewall, permissions, kernel hardening, containers, certificates, and 100+ checks based on CIS Benchmarks, DISA STIG, and industry best practices.

Features

17 Audit Categories

From SSH and PAM to Docker containers and TLS certificates — comprehensive server coverage.

132 Security Checks

Each check includes severity level, detailed findings, and actionable remediation steps.

Standards Based

Aligned with CIS Benchmarks (Ubuntu, RHEL, Debian), DISA STIG, and Lynis audit categories.

Single Binary

Written in Go. No dependencies, no runtime, no agents. Just copy and run.

Multiple Output Formats

Console with colors, JSON for pipelines, YAML for config management integration.

Server Profiles

Pre-built profiles for web servers, database servers, container hosts, and minimal installs.

Quick Start

# Download and run
curl -sL https://github.com/ivan/infraudit/releases/latest/download/infraudit -o infraudit
chmod +x infraudit

# Run full audit (requires root for most checks)
sudo ./infraudit audit

# Run a specific category
sudo ./infraudit audit --category auth

# Use a server profile
sudo ./infraudit audit --profile web-server

# Skip checks you've accepted
sudo ./infraudit audit --skip HARD-007,SVC-012

# List all 132 available checks
./infraudit list

# Export as JSON for CI/CD
sudo ./infraudit audit --format json --output report.json

# Export as YAML
sudo ./infraudit audit --format yaml --output report.yaml
